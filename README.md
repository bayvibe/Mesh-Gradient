# Mesh Gradient Debugger

一个可视化的 Mesh Gradient 调试工具，通过拖拽交互实时创建柔和的多色渐变背景，并导出为可直接使用的 CSS 代码。

## 解决什么问题

CSS mesh gradient 通常需要反复调试颜色、位置、模糊半径和透明度，手动编写代码效率极低。这个工具把调试过程变成可视化操作——在画布上直接拖拽颜色点，所见即所得，调好后一键复制代码。

## 功能

- **可视化编辑** — 点击画布添加颜色点，拖拽调整位置
- **实时调色** — 每个点支持 HSV 取色器、模糊半径、不透明度调节
- **图层排序** — 拖拽左侧列表调整颜色点的层叠顺序
- **复制颜色点** — 快速复制已有的颜色点
- **预设渐变** — 内置 4 款精选渐变，一键加载
- **代码输出** — 实时生成可直接使用的 HTML+CSS 代码
- **下载 PNG** — 将渐变导出为 PNG 图片
- **全屏预览** — 沉浸式查看渐变效果

## 使用方法

1. **添加颜色点** — 点击右侧画布空白区域
2. **调整颜色** — 点击画布上的颜色点或左侧列表项，打开编辑面板
3. **移动位置** — 拖拽画布上的颜色点
4. **调整图层** — 在左侧列表中拖拽颜色点上下移动，改变层叠顺序
5. **复制代码** — 点击底部「复制 CSS」按钮
6. **导出图片** — 点击「下载 PNG」按钮

## 技术实现

使用 div + CSS `filter: blur()` 实现高斯模糊渲染，确保颜色边缘柔和、多色交融自然。输出的代码基于同一方案，复制即可直接使用。
